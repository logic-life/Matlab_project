# **高通滤波器**
```
%sample
n=0:0.01:2;
Rp=0.5;
[z,p,k]=cheb1ap(6,Rp);
[b,a]=zp2tf(z,p,k);
[H,w]=freqs(b,a,n);
subplot(2,1,1);
plot(w,abs(H).^2);
xlabel('w/wc');
ylabel('|H(jw)|.^2');
title('切比雪夫低通滤波器I型(wc=1rad/s)');
[bt,at]=lp2hp(b,a,0.8);%将低通圆形滤波器变为截止频率为0.8rad/s的高通滤波器
[Ht,wt]=freqs(bt,at,n);%频谱分析
subplot(2,1,2);
plot(wt,abs(Ht).^2);
xlabel('wt/wc');
ylabel('|Ht(jw)|.^2');
title('切比雪夫高通滤波器I型(wc=0.8rad/s)');
```
