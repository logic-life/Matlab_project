# **巴特沃斯带通滤波器**
```
%sample
n=0:0.01:2;
Rs=20;
[z,p,k]=cheb2ap(6,Rs);
[b,a]=zp2tf(z,p,k);
[H,w]=freqs(b,a,n);
subplot(2,1,1),plot(w,abs(H).^2);
xlabel('w/wc');
ylabel('|H(jw).^2|');
title('切比雪夫II型低通滤波器(wc=1rad/s)');
w1=0.8;w2=1.4;
w0=sqrt(w1*w2);Bw=w2-w1;
[bt,at]=lp2bp(b,a,w0,Bw);
[Ht,wt]=freqs(bt,at,n);
subplot(2,1,2),plot(wt,abs(Ht).^2);
xlabel('wt/wc');
ylabel('|Ht(jw)|.^2');
title('切比雪夫II型带通滤波器(wc=0.8~1.4rad/s)');
···
